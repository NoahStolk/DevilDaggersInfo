DevilDaggers.info version 5 has been released! This is probably the biggest milestone the project has ever had and I'm very happy about it. After 6 months of working on V5, it is finally time to replace the V4 website (which ran from mid 2018 to early 2022) with newer and better technology.

Almost every feature has been ported to the new website. I'll port the remaining small features soon. This weekend I'll be focusing on getting everything up and running, cleaning up all the old stuff, and fixing any potential issues.

New features

There's some new features too.

  • Graphs for custom leaderboard scores (DDCL 1.2.0.0 and later)
  • Leaderboard history statistics page (WIP)
  • Assets wiki page (WIP)
  • Spawns list is now showing total gems / homing indicator
  • Date picker for leaderboard history
  • This news page
  • Brand new admin portal which I might open to more people soon

I'll be working on more new features later.

Changes

The URL formats have been updated. Most URLs should redirect properly, including all the main pages, and all individual pages for players, spawnsets, custom leaderboards, and mods. Anything else might not redirect (and is probably not important).
