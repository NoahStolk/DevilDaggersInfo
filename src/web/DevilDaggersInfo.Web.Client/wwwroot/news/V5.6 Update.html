New Features

  • Added "Show more waves" button to spawns wiki page.
  • Multiple distributions per tool are now supported. Support for Windows 7 is now distributed as a separate build.
  • World record data can now be sorted.

Changes

  • The Pacifist custom leaderboard category now allows damaging enemies without killing them.
  • Player maintainers can now link users to players.
  • Updated behavior for Skull II.
  • Removed temporary testing feature to parse replays. (This will probably be back in the form of an app at some point.)

Fixes

  • Fixed not applying wave count setting to custom spawnsets.
  • Fixed case sensitive sorting for custom leaderboards, mods, and spawnsets.
  • Fixed not using correct game version selection component on the enemies wiki page.
  • Fixed not formatting spawnset shrink and brightness values.
  • Fixed displaying incorrect total dagger counts (from total accuracy) on the leaderboard pages on mobile.
  • Fixed not displaying total deaths on the leaderboard pages on mobile.

Enhancements

  • Improved the way prohibited assets and binaries are displayed.
  • The spawnset arenas are now larger by default and scale properly on mobile devices.
  • The "How To Use Spawnsets" guide has been extended with information on how to verify game installation on Steam.
